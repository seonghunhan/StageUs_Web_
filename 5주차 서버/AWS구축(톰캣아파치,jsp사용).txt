- 다른 사람이 만들어 놓은 서버를 일정 비용을 지불하고 빌려서 사용하는것,
한국에선 cafe24, 전세계에서 유명한 AWS등 여러 호스팅 업체가 있다.

리눅스란?
처음 서버를 호스팅 받으면 아무것도 없는 깡 기계 덩어리
이 기계 덩어리를 서버로서 동작하게 하려면 운영체제를 설치해줘야 함
대표적인 운영체제로는 WIndows, Mac OS, Android 등이 있음
리눅스는 서버에 특화되어 있는 운영체제

운영체제 : 사용자의 입력이랑 하드웨어랑 연결시켜주는 역할



AWS 로그인
가상머신 시작 -> 운영체제 고르기(우분투서버 Ubuntu Server 20.04 LTS (HVM), SSD Volume Type 인텔64비트)
->서버규모설정(무료버전이니 선택지없음 두번째꺼로밖에못함) 메모리1 -> 검토 및 시작 -> 시작하기
-> AWS 로그인할 키등장(가장중요) -> 새 키페어 생성 -> RSA(암호화방법 기본값) -> 키페어이름:Doris
-> 키페어 다운로드(다운로드파일에저장됨 -> 바탕화면 aws키폴더에 뒀음)(이거 깃허브하면 해킹당해서
요금 폭탄맞을수있음) -> 인스턴스 시작 -> 계속 다음 고고
